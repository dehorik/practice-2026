Ресурсы

Подборка источников, которые помогают понять техническую, образовательную и технологическую основу проекта NeoPixel: от документации ESP-IDF и PlatformIO до материалов по драйверам, дисплеям и фотополимерной печати.

Партнерский и образовательный контекст

Организации и проектная деятельность

Московский Политех

Образовательная организация, в логике которой проектная деятельность рассматривается как практико-ориентированный формат формирования инженерных компетенций.

Проектная деятельность

Информационная площадка, связанная с проектным обучением, командной работой, постановкой задач, отчетностью и представлением результатов учебных проектов.

Техническая документация

Материалы для разработки

PlatformIO Documentation

Документация по среде сборки, конфигурации проектов, работе с платами, библиотеками и мониторингом последовательного порта.

ESP-IDF для ESP32-S3

Официальная документация Espressif по фреймворку ESP-IDF, периферии микроконтроллера, FreeRTOS, GPIO, таймерам и сборке проекта.

A4988 Stepper Motor Driver

Datasheet драйвера A4988, необходимый для понимания сигналов STEP, DIR, EN, микрошагов, ограничений по току и требований к питанию VMOT.

LVGL Documentation

Документация по библиотеке графического интерфейса, которая может использоваться для будущей панели управления принтером.

ESP-IDF LCD SPI

Раздел документации Espressif по SPI LCD, полезный для подключения дисплейного модуля и вывода изображений слоев.

NEMA 17 Stepper Motor

Справочный материал по распространенному форм-фактору шаговых двигателей, применяемых в малых станках и 3D-принтерах.

Для понимания технологии

Что стоит изучить дополнительно

Для более глубокого понимания проекта полезно изучить принципы SLA/MSLA-печати, свойства фотополимеров, механику отрыва слоя от пленки, расчет высоты слоя, требования к равномерности UV-засветки и методы калибровки платформы.

Guide to SLA 3D Printing

Обзорная статья о технологии стереолитографии, полезная для понимания физического смысла послойной фотополимеризации и ограничений печатного процесса.
