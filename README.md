## Вариативная часть задания

В качестве вариативной части выполнена практическая реализация технологии
`Python: A Simple Web Server` из подборки
[`codecrafters-io/build-your-own-x`](https://github.com/codecrafters-io/build-your-own-x).

В директории `src/` создан учебный HTTP-сервер на Python без веб-фреймворков:
сервер работает с TCP-сокетами, разбирает HTTP-запросы, обслуживает статические
файлы, поддерживает методы `GET` и `HEAD`, формирует ответы `200`, `400`, `403`,
`404`, `405`, а также содержит творческую модификацию в виде JSON API
`/api/time` и `/api/echo`.

Подробное руководство и описание исследования находятся в
[`docs/variant_task.md`](docs/variant_task.md), журнал выполнения работ - в
[`docs/variant_journal.md`](docs/variant_journal.md).
